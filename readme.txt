Project: RelationExtraction
Author: Yana Ponomarova 
Creation Date: 12/01/2016
Last updated: 18/01/2016 by Yana Ponomarova

Objective: Parse the sentences with different semantic structures with the goal of extracting the 
(verb_lemma, supplier, receiver, theme) that is used to model Oil & Gas Supply Chain Graph from unstructured text.
This prototype is based on the integration of the Stanford NLP API in to Spark treatments.
